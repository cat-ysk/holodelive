# デジタル対戦ゲーム HOLOdeLIVE

## プロジェクト概要

HOLOdeLIVEは、Godotで構築されたホロライブVTuberキャラクターをフィーチャーした2人用デジタルカードバトルゲームです。プレイヤーはポーカー風のカード組み合わせを使用してターン制ラウンドで競い合い、「ライブ」（パフォーマンス）を獲得します。
このプロジェクトは、2人用対戦カードゲームのデジタル版です。  
Steamプラットフォームで、PC環境向けに配信します。CPUと戦うシングルモードとマルチプレイヤーモードがあります。  
Vtuberグループ、ホロライブのアイドルが描かれたカードを複数組み合わせたユニットを作って勝負し、ライブを開催することで勝利を目指します。

## ゲームルール

2人のプレイヤーは、1回のゲームで最大5ラウンドの対戦を行い、3ラウンド先取で勝利します。  
ラウンド中はターン制となり、プレイヤーは自分のターンでいくつかの行動を行います。  
ターン中またはターン終了時に特定の条件を満たすと、各プレイヤーのユニット同士で勝負を行います。  
ユニットの勝負に勝ったプレイヤーがライブを行い、そのラウンドを取得します。  
ゲーム開始時、コイントスで先行プレイヤーを決定します。各ラウンドでは前のラウンドの敗者が先行となります。  
初期手札は山札から1枚ドローします。手札の上限は3枚で、ターン終了時に余ったカードは自宅へ置かれます。  
山札の総枚数は設定によって可変で、概ね60枚前後です。引き分けはありません。

### カード

アイドルが描かれたカードです。各カードは、IDと名前以外にいくつかの属性を持っています。

- スート(Suit) ... カード1枚につき1個設定されています。アイドルの所属を表し、ユニットの強さを決めるために使用されます。ポーカーのスートとして機能します。以下の種類があります。
	- Lovely
	- Cool
	- Hot
	- English
	- Indonesia
	- Staff
- アイコン(Icon) ... カード1枚につき複数個設定されています。アイドルの性格や性質を表し、ユニットの強さを決めるために使用されます。ポーカーのランクとして機能します。以下の種類があります。
	- Seiso
	- Charisma
	- Otaku
	- Vocal
	- Enjoy
	- Reaction
	- Duelist
	- Kusogaki
	- Intel
	- Sexy
	- Alcohol
- プレイスキル(Play skill) ... カード1枚につき0〜1個設定されています。カードを場にプレイする時に発動する能力です。例えば追加でカードをドローしたり、別のカードを出したり、様々な能力を持っています。
- アクションスキル(Action skill) ... カード1枚につき0〜2個設定されています。ターン中、アクションスキルを使えるステップがあり、そこで1枚につき1回任意に発動できる能力です。
- 常時スキル(Auto skill) ... カード1枚につき0〜1個設定されています。カードがフィールド上にある場合、常に効果を発動しています。

### フィールド

ゲームの戦いの舞台となるフィールドは、以下の区画で分けられています。

- 手札(Hand) ... プレイヤーごとに用意されます。相手に見せないようにカードを持ちます。ゲーム開始時に1枚カードをドローします。ラウンドが進んだ時、手札は引き継ぎます。
- 山札(Deck) ... 1つだけ用意されます。ラウンド開始時に全てのカードをシャッフルして裏向きに重ねて置きます。プレイヤーはここから新しいカードをドローすることができます。
- ユニット(Unit) ... プレイヤーごとに用意されます。最大で3枚のカードをプレイすることができます。ラウンドが進んだ時、ユニットは山札へ戻ります。
- 楽屋(Backstage) ... プレイヤーごとに用意されます。最大で1枚のカードをプレイすることができます。この時、カードは伏せてプレイされ、プレイスキルの発動は保留されます。伏せた状態のカードをゲストと呼び、他のカードのスキルの対象になりません。ターン中、ゲストを表返すことができるステップがあり、そこで保留したスキルを使用することができます。ラウンドが進んだ時、楽屋のカードをユニットに移動させてゲームを開始します。
- 自宅(Home) ... 1つだけ用意されます。スキルの影響でユニットから移動してきたり、手札上限を超えたカードがここに集められます。自宅のカードはどのプレイヤーも所有せず、スキルも発動しません。ターン終了時に5枚以上のカードがある場合、古いものから山札へ戻ります。スキルによって自宅のカードを再利用することも可能です。ラウンドが進んだ時、自宅のカードは山札へ戻ります。

### ステップ

1ターンの中にはいくつかのステップがあり、以下の順番で進行します。

- ターン開始ステップ 
	- 自動で山札からカードを1枚ドローします。
- アクションステップ
	- ユニットまたは楽屋に出ているカードのアクションスキルを1枚につき1回発動することができます。
	- ゲスト状態のカードをオープンし、そのカードが持っているプレイスキルを発動します。
- プレイステップ ... カードを1枚、ユニットまたは楽屋にプレイします。ユニットにプレイする場合はプレイスキルを発動します。楽屋にプレイする場合はゲストとなり、プレイスキルは発動されません。
- ターン終了ステップ ... 後述するライブチェックを行い、条件を満たせばラウンドに勝利します。

### ライブチェック

自分のユニットが3枚とも揃った場合、ライブ準備状態となります。
ライブ準備状態のまま、次の自分のターン開始を迎えると、ライブ開催となります。
ライブ開催時に、相手のユニットがライブ準備状態でなかった場合は、自分がそのラウンドを取得します。
ライブ開催時に、相手のユニットがライブ準備状態だった時、ショウダウンが発生します。
ショウダウンでは、自分と相手の場のカードから算出されたランクで勝負を行い、勝った方がラウンドを取得します。  

### ランク

以下のランクは上から順に強くなります。同じ役の場合、先にライブ準備状態になったプレイヤーが勝利します。
全てのスートとアイコンの強さは同等です。

- ミラクル ... 同じアイコンかつ同じスートが3つ
- トリオ ... 同じアイコンが3つ
- フラッシュ ... 同じスートが3つ
- デュオ ... 同じアイコンが2つ
- カジュアル ... 役なし

### スキルシステム

スキルのコストはありません。同時に発動したスキルの効果は概念上のスタックに積まれ、後から積まれたものから解決します。
